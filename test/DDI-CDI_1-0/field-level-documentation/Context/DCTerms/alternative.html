alternative#

Fully qualified datatype name: DDICDIModels::Context::DCTerms::alternative

Definition#

An alternative name for the resource. The distinction between titles and alternative titles is application-specific. (See DCMI Metadata Terms - https://www.dublincore.org/specifications/dublin-core/dcmi-terms/#http://purl.org/dc/terms/alternative.)
