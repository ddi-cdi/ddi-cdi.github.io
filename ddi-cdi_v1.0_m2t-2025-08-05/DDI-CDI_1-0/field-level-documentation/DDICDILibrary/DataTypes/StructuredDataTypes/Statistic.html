Statistic#

Fully qualified datatype name of the UML model DDI-CDI 1.0: DDICDIModels::DDICDILibrary::DataTypes::StructuredDataTypes::Statistic

Definition#

The value of the statistic expressed as a decimal, float and/or double. Indicates whether it is weighted value and the computation base.

- own datatype -
